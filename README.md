# Retail Inventory & Demand Analysis Dashboard

## Project Overview
This project is an interactive Excel dashboard built to analyze retail inventory, sales performance, demand trends, and profitability. The dashboard helps businesses monitor key metrics, identify high-performing products, and make data-driven inventory decisions.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

## Dashboard Features
- Total Sales
- Total Profit
- Inventory Analysis
- Demand Analysis
- Sales Trend
- Profit by Category
- Profit by State
- Sales by Region
- Sales by Store
- Sales by City
- Payment Method Distribution
- Interactive Slicers

## Key Insights
- Identifies top-performing regions and stores.
- Tracks sales and profit trends over time.
- Compares category-wise performance.
- Analyzes payment method preferences.
- Supports inventory planning and demand forecasting.
